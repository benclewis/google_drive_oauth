# GoogleDriveOauth

Thin wrapper around Oauth2 to make logging in with the Google Drive gem easier.

## Installation

Add this line to your application's Gemfile:

    gem 'google_drive_oauth'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install google_drive_oauth
